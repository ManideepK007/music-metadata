# 🎵 Music Metadata & Analytics Dashboard

## 📌 Project Overview

This project is an Excel-based system to manage and analyze music data. It includes structured metadata and an interactive dashboard.

## 🟢 Features

* Organized 150+ songs with metadata (title, genre, language, artists)
* Multi-language dataset (Telugu, Hindi, Tamil)
* Genre classification system (Romantic, Mass, Party, etc.)
* YouTube views-based performance categorization (Hit, Average, Low)
* Top 10 songs ranking system

## 📊 Analysis

* Songs by Genre
* Songs by Language
* Songs by Movie
* Top Singers

## 📈 Dashboard

* Interactive charts (Pie, Bar)
* Slicers for filtering (Language, Genre)
* Summary metrics (Total Songs, Top Genre, Top Language)

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Slicers
* IF Formulas

## 🎯 Purpose

To demonstrate data entry, data analysis, and dashboard creation skills for entry-level roles like Data Entry Operator and MIS Executive.
